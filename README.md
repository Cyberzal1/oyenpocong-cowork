# Belajar Claude Cowork Bersama Oyen Berpocong

A single-page, static HTML guide (in Malay) explaining Claude Cowork to beginners, presented as a scroll-snapping 9-part deck with an animated mascot.

## Tech
- Plain HTML/CSS/JS — no build step or framework.
- Google Fonts (Baloo 2, Space Grotesk, Inter).
- A fixed background-music player (loop, play/pause, volume, mute) using the native `<audio>` element.

## Structure
- `index.html` — the entire page (markup, styles, and scripts).
- `assets/background-music.mp3` — looping background track for the music player.

## Run locally
Just open `index.html` in a browser, or serve the folder with any static file server, e.g.:

```bash
npx serve .
```
